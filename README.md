## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Run Test Suite:  
Run the application for e2e testing.
`npm start`
`npm test`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000`
